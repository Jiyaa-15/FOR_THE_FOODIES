For The Foodies is a simple and responsive recipe website built using HTML and CSS.  
The goal of this project was to practice frontend basics like layout, responsiveness, and clean UI design.


## About the Project

This website displays recipe cards with ingredients, images, and step-by-step instructions.  
While building this project, the main focus was on:
- Structuring HTML properly
- Using CSS Flexbox for layout
- Making the site responsive for different screen sizes
- Improving the visual look with gradients and hover effects


## Features

- Responsive design (works on mobile, tablet, and desktop)
- Gradient header and footer
- Card-based layout for recipes
- Hover effects on images and cards
- Clean and readable content structure


## Tech Stack

- HTML5  
- CSS3 (Flexbox, Media Queries)


## How to Run Locally

1. Clone the repository  
2. Open the `index.html` file in any browser  

No additional setup required.


## Note

This project was created as a learning exercise while practicing frontend development.  
More features and improvements can be added in the future.
